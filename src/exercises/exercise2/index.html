Exercise: Image Carousel

Create an image carousel that allows users to view a series of images. The carousel should include previous and next buttons to navigate between images and display indicators to show the current position in the image sequence. Ensure that the carousel is responsive and functions smoothly on different screen sizes. Bonus challenge: Add auto-play functionality where images rotate automatically with a pause option.
